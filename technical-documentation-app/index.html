Introduction

JavaScript is a programming language used to create dynamic content on websites.

It runs in the browser and allows interaction with users.

Variables

Variables store data values.

let name = "Diana";

Variables can be reassigned.

Functions

Functions group reusable code.

function greet() { console.log("Hello"); }

They help keep code organized.

Loops

Loops repeat code blocks.

for (let i = 0; i < 5; i++) { console.log(i); }

They reduce repetition.

Arrays

Arrays store multiple values.

const numbers = [1, 2, 3];
  • Indexed from 0
  • Can be modified
  • Can hold mixed types
